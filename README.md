## Overview

## Step

### 1. Install homebrew

install from [here](https://brew.sh/)

### 2. Install git

```sh
$ brew install git
```

### 3. Through path

```sh
$ ehoc "export PATH=/usr/local/bin/git:$PATH" >> ~/.
```

```sh
$ source ~/.zshrc
```

### 4. Clone this repository

```sh
$ git clone "xxx"
```

### 5. Run setup script

```sh
$ ./setup_all.sh
```
